# Stand-Alone-LAWbot (Sandbox)

**A prototype sandbox model of the larger LawAI project**

## Overview
This repository contains a sandbox prototype of **LawAI**, our comprehensive AI-powered legal assistant platform developed for Smart India Hackathon 2024. This standalone version serves as a testing and demonstration environment for the core functionality of the larger LawAI ecosystem.

## Project Context
- **Main Project**: LawAI - Comprehensive Legal AI Platform
- **This Repository**: Sandbox/Prototype for testing and review
- **Purpose**: Proof of concept and feature validation
- **Team**: CODE-A-COLA
- **Event**: Smart India Hackathon 2024

## Functionality
This sandbox model focuses on the core feature of **FIR Filing Assistance**:
- Analyzes criminal incident descriptions
- Identifies applicable Indian laws, acts, and sections
- Provides comprehensive legal guidance for law enforcement officers
- Ensures complete and legally sound FIR filing

## Technology Stack
- **Frontend**: HTML5, CSS3, JavaScript, Tailwind CSS
- **AI Integration**: Google Gemini API
- **Backend**: Node.js (hosted on Vercel)
- **Deployment**: Static web hosting compatible

## Features
- Interactive query interface
- Real-time AI-powered legal analysis
- Comprehensive coverage of Indian Constitution and criminal law
- Responsive, professional UI design
- Loading states and user feedback
- Markdown-formatted responses

## Usage
1. Open `index.html` in a web browser
2. Enter a criminal incident description
3. Receive comprehensive legal guidance including applicable acts and sections
4. Use the analysis to file complete FIRs

## Files
- `index.html` - Main application interface
- `test.html` - Development/testing version
- `favicon.ico` - Website favicon
