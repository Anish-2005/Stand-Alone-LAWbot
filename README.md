# LawAI - Intelligent Legal Assistant

**AI-Powered Legal Analysis for Law Enforcement Officers**

## Overview
LawAI is an advanced AI-powered legal assistant platform designed to support law enforcement officers in filing comprehensive and legally sound First Information Reports (FIRs). This application analyzes criminal incident descriptions and identifies all applicable Indian laws, acts, sections, and legal provisions.

## Features
- **Intelligent Legal Analysis**: AI-powered identification of relevant laws and sections
- **Comprehensive Coverage**: Extensive knowledge of Indian Constitution and criminal law
- **Professional UI**: Modern, responsive design with intuitive user experience
- **Real-time Processing**: Instant analysis with loading states and feedback
- **Educational Guidance**: Detailed explanations of legal provisions and their applications

## Technology Stack
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: Tailwind CSS with custom CSS
- **Icons**: Font Awesome
- **AI Engine**: Google Gemini API
- **Backend**: Node.js (hosted on Vercel)
- **Fonts**: Inter font family

## Usage
1. **Access the Application**: Open `index.html` in a modern web browser
2. **Describe the Incident**: Enter a detailed description of the criminal incident
3. **Get Legal Analysis**: Receive comprehensive legal guidance including:
   - Applicable acts and sections
   - Detailed explanations of legal provisions
   - Relevance to the specific incident
   - Guidance for FIR filing
4. **Review and Apply**: Use the analysis to ensure complete FIR documentation

## How It Works
1. **Input Processing**: User describes a criminal incident in plain language
2. **AI Analysis**: Google Gemini API analyzes the description against comprehensive legal database
3. **Legal Mapping**: Identifies all relevant Indian laws, acts, and sections
4. **Detailed Guidance**: Provides explanations, significance, and application context
5. **FIR Support**: Ensures law enforcement officers have complete legal information
